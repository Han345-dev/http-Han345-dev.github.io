CHAPITRE VII: LES TABLEAUX


I/ INTRODUCTION

Les tableaux peuvent servir de grille qui contiennent des données structureés en ligne et en colonne comme par exemple une liste d'article avec leur prix mais il peuvent également servir de conteneurs qui renferment d'autres objets HTML afin de structurer sa page web sous forme de cellule et et faciliter ainsila mise en forme.
Les tableaux HTML peuvent servir à structurer sa page web.

Pour créer un tableau trois balise sont nécessaires
  • < table >..< /table >
  • < tr >..< /tr >
  • < td >..< /td >

II/ LA BALISE < table >..< /table >

La balise table est le conteneur principal, Elle permet de déclarer un tableau et peut accueillir de nombreux attributs dont les plus fréquents sont:

  1. Border
  2. Cellspacing
  3. Cellpadding
  4. Width
  5. Height
  6. Bgcolor

II/ LA BALISE < tr >..< /tr >

La balise < tr >..< /tr > permet d'jouter une ligne à un tableaux, Elle est directement déclarée dans la balise < table >..< /table >.

III/ La balise < td >..< /td >

La balise < td >..< /td > permet de créer une colone. Elle est déclarée dans la balise < tr >..< /tr >.

IV/ LA BALISE < caption >..< /caption >

La ballise < caption >..< /caption > est déclarée directement aprés la balise < table >..< /table >. Elle contient le titre du tableau et elle peut accueillir l'attribut align qui peut prendre une des valeurs suivantes: Top ou Bottom.

V/ LA BALISE < thead >..< /thead >

Cette balise permet de déclarer les cellules d'entete. Elle est déclarée dans la balise < table >..< /table > et peut contenir des lignes qui contiennent des colonnes.

VI/ LA BALISE < th >..< /th >

Quand la balise < thead >..< /thead > est déclarée, Il est préférable de remplacer ses balises < td >..< /td > par < th >..< /th >.
La balise < th >..< /th > désigne donc une legende pour les colonnes d'entete.

VII/ LA BALISE < tbody >..< /tbody >

il s'agit d'une balise qui est censée accueillir les cellules du corps du tableau. Elle est déclarée dans la balise < table >..< /table > et peut contenir des balises < tr >..< /tr > et < td >..< /td >

VIII/ LA BALISE < tfoot >..< /tfoot >

IL s'agit d'une balise qui est censée accueillir les cellules du pied du tableau. Elle est déclarée dans la balise < table >..< /table > après la balise < tr >..< /tr >. Elle peut contenir les balises < tr >..< /tr > et < td >..< /td >

IX/ LE FUSIONNEMENT DES CELLULES

On peut fusionner les cellules horizontalement ou verticalement. Mais le fréquent c'est de fusionner horizontalement grace à l'attribut colspan.

1- L'attribut colspan. colspan est inclus dans la balise de colonne < td >..< /td >. Il indique indique à celle ci combien de colonnes il faut fusionner pour avoir une à la fin.

2- L'attribut rowspan

rowspan quant à elle fusionne à la verticale. Le principe est le meme.
