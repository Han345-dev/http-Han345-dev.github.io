CHAPITRE V: LES PARAGRAPHES, LES CONTENEURS, ET LES LISTES


I/LES PARAGRAPHES

La balise < p >..< /p > de type block c-à-d qu'elle crée un block et enregistre automatiquement un retour à la ligne. Elle sert à définir un paragraphe. On peut la doter de l'attribut 'align' qui permet d'aligner à sa guise le contenu du paragraphe. Les différentes valeurs de 'align' sont:
  • Left (gauche)
  • Right ( droite)
  • Jutify ( justifie le contenu)
  • Center (centre le paragrphe)

II/ LES CONTENEURS

Les conteneurs sont des balises qui peuvent renfermer d'autres éléments comme du texte ou des images, Les conteneurs les plus célèbres sont les balises < div >..< /div > et < span >..< /span >

1- La balise < div >..< /div >

La balise < div >..< /div > est une balise de type block. Elle permet de définir un conteneur ou un block qui contien d'autres éléments,Elle sert généralement à mieux diviser la page web pour placer le bon contenu au bon endroit. Cependant la balise < div >..< /div > nécéssite des styles CSS pour qu'elle sous-titresdotée de toute sa force. Les attributs dont on peut la dotée sont les memes que ceux de < p >..< /p >.

2- La balise < span >..< /span >

C'est aussi un conteneur mais de type inline, Elle occupe juste assez d'espace pour afficher ce qu'elle contien et pas toute la ligne comme la balise < div >..< /div >. Par consequent elle n'engendre pas de retourà la ligne.

III/ LES LISTES HTML

Pour afficher plusieur entrées, l'une en dessous de l'autre il y a pas mieux que les listes. Il y en a d'autres type (listes ordinnées et listes non ordonnées).

1- Les listes ordonnées

La balise < ol >..< /ol > permet d'avoir les listes ordonnéesou numérotées. Elle contient une autre ablise < li >..< /li >. Celle ci permet de définir Une entrée de la liste.
La balise < ol >..< /ol > peut avoir des attributs dont les plus importantes sont:

  • Type: sert à définir le type de marqueur de la liste, Les valeur de l'attribut sont '1' pour les chiffres classique, 'A' pour l'alphabet majuscul, 'a' pour l'alphabet miniscul, 'I' pour les chiffres romains majuscul, et 'i' pour les chiffres romains miniscul.

  • Start: indique la valeur du marqueur du début.


2- Les listes non ordonnées

La balise < ul >..< /ul > permet de créer les listes non ordonnées, Elle contient aussi la balise < li >..< /li > qui permet d'ajouter d'autres éléments à la liste tout comme la balise < ol >..< /ol >.
La balise < ul >..< /ul > pocède des attributs qui permettent de personnaliser la liste. Le plus important d'entre eux est l'attribut type qui définit le type de marqueur et qui peut avoir les valeurs suivantes:

  • Disc: le marqueur est un cercle plein.

  • Circle: le marqueur est un cercle.

  • Square: le marqueur est un carré plein.
