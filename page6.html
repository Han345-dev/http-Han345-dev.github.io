CHAPITRE VI: LES LIENS


I/ LES LIENS HYPERTEXTES

1- Définition Un lien hypertexte ou hyperlien est un objet HTML (texte, image...) sur lequel on peut cliquer pour aller vers une autre page ou un autre emplacement dans la meme page. Il est considéré comme l'un des éléments les plus importantesdu langage HTML puisqu'il permet de relier les pages les unes aux autres pour constituer ainsi le web tel que nous le connaissons.

2- La ballise < a >..< /a >

La balise < a >..< /a > permet de créer un lien hypertexte pointant vers un autre document ou autre emplacement. Un lien classique est réconnaissable par sa couleur bleue et un style souligné ( ces proprietés peuvente etre changées à l'aide du langage CSS).

La balise < a >..< /a > dispose des attributs suivants:

  • href: Désigne le chemin vers la page de destination après avoir cliqué sur le lien.

  • target: Désigne où la page de destination sera ouverte Il peut contenir plusieur valeurs mais une seule est fréquemment utilisée, Il s'agit de 'blank'.
    La valeur blank signifie que le lien sera ouvert dans un nouveau onglet. C'est utile pour les liens pointants vers un autre site web, Si l'attribut target n'est pas spécifié alors la page de destination sera ouverte dans le meme onglet.

  • Title: Ecrit un message qui sera affiché dans une information bulle si on survole le lien avec la sourie.
