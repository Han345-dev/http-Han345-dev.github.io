CHAPITRE I: INTRODUCTION









HTML est un langage de balissage utiliser pour structurer le contenue d'une page web. Il permet de définir des éléments tels que les titres, les paragrphes, les images, les liens, et bien plus encore. Les balises html sont des éléments entourés de chevrons, comme < html >...< /html >, < body >...< /body >, < h1 >...< /h1 > etc.

ces balises permettent aux navigateurs web de comprendre et d'afficher le contenu de manière approprié.
