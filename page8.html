CHAPITRE VIII: LES FORMULAIRES HTML

I/ INTRODUCTION

Un formulaire HTML permet de rendre la page plus intéractive en la rendant capable de dialoguer avec l'utilisateur a trvers des champs de saisie et boutons. Il permet à l'utilisateur de valider un choix ou encore de saisire du texte.
Ces informations seront ensuite envoyées au serveur pour les traiter.

II/ LA BALISE < form >..< /form >

Elle délimite les champs du formulaire, Elle peut avoir plusieurs attributs dont les plus fréquents sont:

1- L'attribut name

L'attribut name d'identifier le formulaire par un nom.

2- L'attribut method

L'attribut method permet de spécifier la methode à l'utilisateur pour envoyer les formulaires au serveur. En effet l'objectif d'un formulaire c'est d'envoyer les informations saisies par client au serveur.
En HTML il existe deux methodes pour envoyer un formulaire GET et POST.
  • La methode GET: permet d'envoyer les données par url

  • La methode POST: permet d'envoyer les données du formulaire au serveur a trvers une entete.


3- L'attribut action

Quand un formulaire est remplit par l'internaute il est posté au serveur pour le traitement. IL faut donc désigner la page qui se chargera du traitement des données.
Cette page est spécifiée dans l'attribut action.

II/ LA BALISE < input >

1- Définition

La zone de texte est le champ de formulaire le plus célèbre, Il permet d'ecrir un texte. Pour créer une zone de texte on fait appel à la balise < input >.
C'est une balise orpheline donc pas de balise fermante.

2- L'attribut type

L'attribut type permet de définir le type de champ à intégrer. Voici les valeurs à mettre pour cet attribut et qui permettent d'avoir ls champs les plus fréquents:

  • texte
  • password
  • radio
  • chekcbox
  • file
  • hidden
  • button
  • reset
  • submit

III/ LA BALISE < textarea >..< /textarea >

Cette balise permet de définir un espace de texte. IL dispose des attributs suivats : Cols pour la largeur et rows pour la hauteur.

IV/ LA BALISE < select >..< /select >

ELle permet de faire sotir Une liste de sélection ou une list edéroulante, Pour remplire cette liste avec des valeurs appelées 'option' on déclarer une autre balise à l'intérieur c'est la balise < option >..< /option >, Dans celle ci on déclare le mot à afficher en tant qu'élément de la liste.

VI/ LA BALISE < fieldset >..< /fieldset >

Elle permet d'appliquer un cadre qui entoure le formulaire.

VII/ LA BALISE < legend >..< /legend >

Elle permet de donner la legende du formulaire.
