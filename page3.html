CHAPITRE III: LA STRUCTURE D'UNE PAGE HTML


Le code source de la page HTML est structuré de cette manière:


< html >

< head >

< title > Le titre ici< /title >

< /head >

< body >

Le corps ici

< /body >

< /html >

I/ L'ENTETE

la balise < head >......< /head > regroupe les informations qui ne sont pas affichées sur la page HTML.
EX: Le titre de la page; Les informations utilisées par les moteurs de recherche.

II/ LE CORPS

< body >.......< /body > regroupe les informations qui s'affichent dans le navigateur.
