CHAPITRE II: NOTION DE BALISES

I/ DEFINITION

Une balise est une suite de caractères commençant par "<", et finissant par ">", permetant de donner des indications sur la signification, le formatage ou les caractéristiques des différents éléments du contenue d'une page web.


II/ LES TYPES DE BALISES

1- Les balises < ouvrante > et < /fermante >

La balise < ouvrante > marque le debut d'un traitement, et la balise < /fermante > marque la fin de ce traitement.
SYNTAXE: < balise > ..................< /balise >.
NB: Chaque balise ouvrante on associe une balise fermante.

2- Les balises orpheline Une belise orpheline en HTML est une balise qui n'a pas de fermeture correspondante.
EX: < br/ >
< img >
< hr >

II/ LES ATTRIBUTS DES BALISES

Les balises HTML peuvent souvent etre accompagnés d'attributs qui fournissent des informations suplementaires sur l'élément. Les attributs sont spécifiés dans la balise d'ouverture séparé par un signe égal.
SYNTAXE: < balise attribut="valeur" >.
EX : < a href="http//www.google.fr" >....< /a >.
